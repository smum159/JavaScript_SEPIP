ARRAY STRUCTURE

An array as any variable is declared with var or let, the difference to any other variable is that the data stored in an array is between brackets “[ ]” and inside of these separated with commas a different the data what meant to be stored.

ACCESSING TO ELEMENTS

To access to certain elements of an array it only matters to know the position where the element is, in the arrays the position of the elements start with 0 being the first position and so on. These elements can also be stored within variables for the use that they are required to give.

POP & SHIFT

When it comes to removing elements of an array one by one there are two methods, POP and SHIFT the difference between these two is that POP work with the arrays as a pile, removing the last element in the array, or in this example. By the other side, SHIFT removes the first element in the array.
If the method is used with a variable this one will take the element removed as a value.

PUSH

Just as POP work the arrays as piles PUSH do the same thing, but this time it will add a new element to the last position of the array.

CHANGING

If what you want could be to add an element in a certain location of you could use change the element for another one, keep in mind that change will work as a substitution of the current element for the one you want to push in the array. The only thing is, in the same way, you access to any element in an array and changing his value.

DELETE

In the same way, as change an element in the array we can delete it using the method delete, in this case, this method doesn't use brackets (), the word goes before we access to the element we want to remove. You must have in mind that this method doesn't fully delete the element, but just clean the position selected leaving it empty.

SPLICE

This method will help us to add new elements and/or delete elements from an array. The method splice() use many parameters, the first two should be numbers, the first one represents the position where the splice will start to work and the second one represent how many elements will be removed from the position specified in the last parameter, after that the rest of the parameters will be all the elements you will add to the array.

SLICE

As his name implies this method slice an array starting from a certain position, but it doesn't separate the slice from the original array but creates a new array using a copy from the sliced elements. To work correctly with the array's slice you must save the result in a variable.

CONCAT

When it comes to concat an array it means to join two arrays into one, but it doesn't mean to delete the previous arrays. As the same of slice, we must save the result of the method into a variable if we want to use it later. Have in mind that the array that will be merged into the another their elements will start after the elements of the array that is merged.

SORT

Sort as his name implies it sorts the elements of an array changin their positions, but it only sort alphabetically, no matter if you have strings or numbers.
