Variables

JAVASCRIPT VARIABLES

javascript has the following types of variables:

  • Text
  • Numeric
  • Booleans
  • Arrays
  • Objects

TEXT TYPE VARIABLES

Text variables store a string of characters.

You can use single quotes or double quotes

The use of single quotes or double quotes is indistinct. The important thing is the order of how we use them. If, for the needs of the program, we ever need to use single quotes, then we would use single quotes for the text type variable. And vice versa. The only thing that is not allowed is to mix Single quotes and close them with double quotes. And vice versa.

NUMERIC TYPE VARIABLES

Numeric type variables store numbers.

Each number, even integer numbers, decimal numbers... have no diference so they all are numeric variables for JavaScript. However, advanced programming requires special attention for handle decimal numbers.

This type of variables do not require any kind of quotes.

BOOLEAN TYPE VARIABLES

Boolean type variables store TRUE or FALSE values.

ARRAY TYPE VARIABLES

Array type variables are a special type of object that stores an inventory of ordered contents that can be of any type of variable.

The array can store text, numbers, another arrays and any type of variable.

Also known as vectors or lists.

Objects

An object is a standalone entity, with properties and type. Compare it with a cup, for example. A cup is an object, with properties. A cup has a color, a design, weight, a material it is made of, etc. The same way, JavaScript objects can have properties, which define their characteristics.

So we can get a specific value we have to follow the next structure: «persona.nombrePropiedad»

An important thing about objects, objects do not need and specific order of their attributes.

each attribute has to be separed by a comma «,» for exeption the last one.
